# Cine

Cine is an Native Android application built with Kotlin and Jetpack Compose that showcases movies
and TV shows using the TMDB (The Movie Database) API.

## Tech Stack

* Kotlin.
* Jetpack compose
* Material design 3.
* Clean architecture.
* Coroutines, StateFlow.
* Hilt.
* Retrofit.
* Coil.
* DataStore.
